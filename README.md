## 🚀 Локальное развертывание

### Предварительные требования
Для запуска проекта на вашем компьютере должны быть установлены:
* [Python 3.10+](https://www.python.org/downloads/)
* [Docker и Docker Compose](https://www.docker.com/products/docker-desktop/)

### Шаг 1. Настройка окружения и установка зависимостей
1. Склонируйте репозиторий и перейдите в папку проекта:
   ```bash
   git clone <ссылка_на_ваш_репозиторий>
   cd <имя_папки_проекта>
   ```

3. Создайте виртуальное окружение:
   ```bash
   python -m venv .env
    ```

4. Активируйте виртуальное окружение:
    ```
   * **Windows:** `.env\Scripts\activate`
   * **Linux/macOS:** `source .env/bin/activate`
   ```

5. Установите необходимые библиотеки:
   ```bash
   pip install -r requirements.txt
    ```

### Шаг 2. Запуск базы данных (PostgreSQL)
Поднимите контейнер с базой данных в фоновом режиме с помощью Docker Compose:
```bash
docker compose up -d
```

### Шаг 3. Инициализация схемы базы данных

> **Примечание:** Оставьте в файле только тот вариант инициализации схемы, который вы реально используете в коде.


### Шаг 4. Запуск приложения
После успешной настройки базы данных запустите основной файл проекта:
```bash
python main.py
```

---

**Полезные команды:**
* Остановить базу данных: `docker compose stop`
* Полностью удалить контейнер БД: `docker compose down`
* Деактивировать виртуальное окружение Python: `deactivate`
